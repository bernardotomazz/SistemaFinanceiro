# Sistema Financeiro Pessoal

API REST desenvolvida com FastAPI para gerenciamento de receitas e despesas pessoais.

## Tecnologias Utilizadas

### Backend

- Python

- FastAPI

- SQLAlchemy

- SQLite

### Frontend

- React

- TypeScript

- Tailwind CSS

## Estrutura do Projeto

```text
SistemaFinanceiro/
├── backend/
│   ├── routers/
│   ├── services/
│   ├── schemas/
│   ├── database/
│   └── main.py
│
├── frontend/
│   ├── src/
│   │   ├── components/
│   │   ├── pages/
│   │   └── layouts/
│   └── package.json
```
